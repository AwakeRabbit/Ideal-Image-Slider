Multiple Sliders

Multiple sliders are supported and can be controlled independantly.

  • JS
  • HTML
var slider1 = new IdealImageSlider.Slider('#slider1');
var slider2 = new IdealImageSlider.Slider({
    selector: '#slider2',
    effect: 'fade'
});
